# ionic4-news-app
Membangun aplikasi berita dengan Ionic 4

1. Install node.js

2. Install ionic & cordova
   npm install -g cordova ionic

3. Dari CMD pindah ke folder project ionic : 
   npm install

4. Menjalankan ionic dari browser : 
   ionic serve
